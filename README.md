✈️ Flight Data Analysis & Analytics Dashboard
📌 Project Overview
This project performs Exploratory Data Analysis (EDA), Statistical Analysis, SQL-based insights, and an interactive dashboard on airline flight data.
The goal is to extract meaningful business insights from the dataset and visualize them through a Streamlit web application.
The project simulates a real-world data analyst workflow, starting from raw data exploration to building a data-driven dashboard.
📊 Project Workflow



1️⃣ Exploratory Data Analysis (EDA)
Performed detailed EDA using Python libraries to understand patterns in flight data.
Key analysis included:
Flight price distribution
Airline-wise price comparison
Arrival and departure time analysis
Stops vs price relationship
Duration vs price trends
Libraries used:
Pandas
NumPy
Matplotlib
Seaborn


2️⃣ Statistical Analysis
Performed statistical testing to validate patterns in the dataset.
Methods used:
Hypothesis Testing
ANOVA tests
T-tests
Confidence Interval estimation
Correlation analysis


This helped determine whether factors such as arrival time, airline, or number of stops significantly impact flight prices.





3️⃣ SQL Analysis
Used MySQL to query and analyze the dataset.
Key SQL tasks included:
Finding busiest airports
Calculating airline frequency
Identifying popular routes
Aggregating flight counts by city
Performing grouped analytical queries
Example insights:
Most frequent airlines
Cities with highest flight traffic
Flight distribution across routes



4️⃣ Flight Analytics Dashboard
Built an interactive dashboard using Streamlit to visualize airline data.
Dashboard features:
Search flights between cities
Airline market share visualization
Busiest airport analysis
Interactive data tables
Dynamic charts using Plotly
Users can explore flight data directly through the web interface.

📈 Key Insights
Some insights derived from the analysis:
Flight prices vary significantly depending on arrival time and number of stops.
Longer flight durations generally lead to higher ticket prices.
Certain cities act as major flight hubs, showing higher traffic frequency.
Airline market share distribution highlights the dominant carriers in the dataset.
